N-queens game

This game is based on the 8-Queens Problem and the challenge is to place N queens on an N x N chessboard so that no two queens attack each other.

Note that every time you place a queen on the chessboard, a solver checks if there is a solution for the current placement, so there might be a minimal delay after each step. You can find the code for this game on https://github.com/angee/nqueens-game.
