I know you as someone very brainy who loves a puzzle. This is why I programmed a little "puzzle-game" as a present for you - the N-queens game!

The game is based on the 8-Queens Problem and the challenge is to place N queens on an N x N chessboard so that no two queens attack each other. Note that every time you place a queen on the chessboard, a solver checks if there is a solution for the current placement, that's why there is a minimal delay after each step. Sorry for the simple graphics, that's not my strength ;-)
